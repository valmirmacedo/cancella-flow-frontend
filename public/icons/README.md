# Para gerar os ícones PNG a partir do SVG, você pode usar:
# 1. Ferramentas online como https://cloudconvert.com/svg-to-png
# 2. Ou usar ImageMagick: convert icon.svg -resize 72x72 icon-72x72.png
# 3. Ou usar um gerador de PWA como https://www.pwabuilder.com/

# Tamanhos necessários:
# icon-72x72.png
# icon-96x96.png  
# icon-128x128.png
# icon-144x144.png
# icon-152x152.png
# icon-192x192.png
# icon-384x384.png
# icon-512x512.png

# Por enquanto, você pode usar o ícone SVG ou gerar os PNGs conforme necessário
